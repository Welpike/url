How to use my redirect .js file?

First : include this script in your <head>.

<script src="https://welpike.github.io/cdn/redirect.js"></script>

Secondary : create a script (file or in <script>) after the script include tag.

<script>
let url = "https://welpike.github.io/docs.html"; // define the url
delay = 5; // define the time before the redirecting
redirect(url, delay); // call function
</script>

How does it works ?

This is the source code of redirect.js.

// Welpike - welpike.github.io
function redirect(url,delay){ // get variables
url = new URL(url); // transform url into a "url string"
setTimeout(
"document.location.replace(url)", // function that replace the url of the window by the one who we define before
delay + '000', // convert the delay : milliseconds to seconds
);
}
